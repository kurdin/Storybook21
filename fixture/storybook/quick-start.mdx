# Quick start


#### Install
```bash
npm i --save-dev storybook21
```
<br />

#### Initialization
```bash
npx storybook21 init
```
<br />

The command above will make the following changes to your local environment:
- 🏝️ create playground directory
- 📦 create storybook directory
- ⚙️ create config file
- 🚀 added start and build script to your package.json

<br />
#### Run app and go to <a href="http://localhost:4242" target="_blank">localhost</a>
```bash
npm run start:storybook
```
