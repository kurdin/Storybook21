#### How show prop types?
We provide <code>PropTypes</code> component.
For show props you need to specify path to component in <code>component</code> field in object in navigation.
Use props _displayName_ for show props-table. If _displayName_ is incorrect you'll get console.error.

```tsx
<PropTypes displayName="YOUR_COOL_COMPONENT_NAME" />
```
Under the hood we use [react-docgen-typescript](https://github.com/styleguidist/react-docgen-typescript) for typescript files and [react-docgen](https://github.com/reactjs/react-docgen) for javascript files.

Show examples [ts props](/components/ts-props-example) or [js props](/components/js-props-example)
